Working with Arrays

Looking at built-in methods of arrays

Arrays are zero index, they start counting from zero.

Push-add the last value in an array

shift- method used to remove the first value in an array

unshift- Method used to add value on index 0

pop- method used to remove the last value in an array

Concat- Is a method that is used to merge arrays

indexOf - Get the index of a value in array

typeof - Get the type of data in an array
